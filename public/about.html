About Justice Connect

Justice Connect is an open-source court complaint management demo designed to simplify e-Filing, case tracking, and resolving court disputes. It demonstrates:

  • Session-based authentication with hashed passwords
  • File uploads with `multer`
  • SQLite persistence and simple migration handling
  • Admin workflows for status updates and remarks
